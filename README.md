# Diabetes Health Data Analysis

Exploratory data analysis on the PIMA Indians Diabetes Dataset 
to identify patterns associated with diabetes risk.

## Key Findings
- Diabetic patients have ~30 units higher average glucose levels
- Diabetes rate peaks in the 50s age group
- ~49% of Insulin values are missing (recorded as zero)
- Glucose is the strongest predictor of diabetes outcome

## Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook

## Dataset
PIMA Indians Diabetes Dataset — Kaggle
